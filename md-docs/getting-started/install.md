### DigitalOcean LEMP

Follow these steps if the development team is big and across different timezones.

Requirements:
  - LEMP environment
  - PHP >=5.6.4

```
apt-get update
apt-get install git node
composer global require "laravel/installer"
npm install
composer install
gulp --production
```
